hAI.AnythingLLM

Sauberes Repository für AnythingLLM mit Docker Compose, Portainer-Stack, persistenter Storage-Struktur, Dokumentation und vorbereiteter Projektseite für den schnellen Upload nach GitHub.

Jetzt installieren Repository öffnen
Compose
Startet AnythingLLM lokal mit persistentem Storage unter ./storage.
Portainer
Stack-Datei mit Host-Pfad /opt/hAI.AnythingLLM/storage.
Troubleshooting
Hinweise für Rechteprobleme, SQLite-Fehler und UI-Fehler wie „Failed to fetch“.

1. Repository

Repository klonen und in das Projektverzeichnis wechseln.

git clone https://github.com/jbkunama1/hAI.AnythingLLM.git
cd hAI.AnythingLLM

2. Storage

Datenverzeichnis anlegen und die korrekten Rechte setzen.

mkdir -p ./storage
sudo chown -R 1000:1000 ./storage
chmod -R 775 ./storage

3. Start

AnythingLLM mit Docker Compose oder per Portainer Stack starten.

docker compose up -d
